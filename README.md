# vscode-stock

支持在状态栏显示指定股票行情

![img](https://sm.ms/image/1PdsH4mfF9YTBWi)

## 配置

> 文件 > 首选项 > 设置 > 扩展 > statusbar股票插件
>
> 支持修改股票代码和更新间隔时间
>   * 股票代码支持json数组
>   * 更新间隔时间需重启vscode
>
> ![img](https://sm.ms/image/o7iNJTv6qk1buQy)
>
> ![img](https://sm.ms/image/Y13XzEWbNpJeFPO)

## 隐藏股票

在状态栏点击某支股票可隐藏该股票信息，暂`不支持`恢复，重启后可恢复原状态。
